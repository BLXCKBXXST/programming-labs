## Задание (вариант 1)

Дана прямоугольная матрица A размера N × M. Требуется:

1. Найти **позицию максимального элемента** в A.
2. Сформировать матрицу **B** — это «минор»: A с вычеркнутой строкой и столбцом, где находится максимальный элемент. Размер B = (N − 1) × (M − 1).
3. Вывести B на экран.
4. Посчитать, сколько в B **строк, содержащих хотя бы один ноль**.
5. Вывести **главную диагональ** B (точнее — её квадратную часть min(N−1, M−1) × min(N−1, M−1)) в **обратном порядке**.

## 🧪 Пример

Ввод (3×3):

```
3 3
1 2 3
4 9 5
6 7 8
```

Максимум `9` в позиции (1, 1). Вычеркивая строку 1 и столбец 1:

```
B = | 1 3 |
    | 6 8 |
```

Вывод:

```
1 3
6 8
0          ← нет строк с нулями
8 1        ← главная диагональ в обратном порядке
```
