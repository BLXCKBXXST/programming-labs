## Задача 13 — Степень числа 2 (`1zadanie.c`)

### Условие

Пусть имеется некоторое натуральное число `n`. Проверить, является ли оно степенью числа 2, то есть найдётся ли такое целое неотрицательное число `k`, что `n = 2^k`.

### Метод решения

Степень двойки в двоичном представлении имеет ровно один единичный бит. Например:

| Число | Двоичное представление | Степень двойки? |
|-------|------------------------|------------------|
| 1     | `00000001`             | ✅ (2⁰)          |
| 2     | `00000010`             | ✅ (2¹)          |
| 4     | `00000100`             | ✅ (2²)          |
| 8     | `00001000`             | ✅ (2³)          |
| 6     | `00000110`             | ❌               |
| 7     | `00000111`             | ❌               |

Ключевое свойство: для любой степени двойки `n & (n - 1) == 0`.

Почему это работает:
- `n` в двоичном виде: `0001000`
- `n - 1` в двоичном виде: `0000111`
- `n & (n-1)` = `0000000` → результат **0** → число является степенью двойки

Если число **не** степень двойки (например, `6 = 0110`):
- `6 - 1 = 5 = 0101`
- `6 & 5 = 0100` → результат **не 0** → не степень двойки

### Используемые битовые операции

- `&` — побитовое И

### Пример работы

```
Введите натуральное число n: 8
8 является степенью числа 2

Введите натуральное число n: 6
6 не является степенью числа 2
```

## Задача 14 — Количество единичных битов (`2zadanie.c`)

### Условие

Найти количество единичных битов в целом неотрицательном числе `a`.

### Метод решения

Используется **алгоритм Брайана Кернигана**: операция `a &= (a - 1)` сбрасывает (обнуляет) самый младший единичный бит числа. Повторяя эту операцию до тех пор, пока число не станет равно нулю, и считая количество итераций, получаем количество единичных битов.

Почему `a & (a - 1)` сбрасывает младший единичный бит:
- `a = 1100` (двоичное)
- `a - 1 = 1011`
- `a & (a-1) = 1000` — младший единичный бит сброшен

### Пример работы по шагам для числа 13 (`1101` в двоичном):

| Шаг | a (двоичное) | a-1 (двоичное) | a & (a-1) | Счётчик |
|-----|-------------|----------------|-----------|--------|
| 1   | `1101`      | `1100`         | `1100`    | 1      |
| 2   | `1100`      | `1011`         | `1000`    | 2      |
| 3   | `1000`      | `0111`         | `0000`    | 3      |

Результат: 3 единичных бита.

### Используемые битовые операции

- `&=` — побитовое И с присваиванием

### Пример работы программы

```
Введите неотрицательное целое число a: 13
Количество единичных битов в числе 13: 3

Введите неотрицательное целое число a: 255
Количество единичных битов в числе 255: 8
```
