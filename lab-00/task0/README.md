# Задание 0 — hello, world

Минимальная программа на C. Цель — отработать цикл «создать файл → скомпилировать → запустить».

## Код

[hello.c](hello.c):

```c
#include <stdio.h>

int main() {
    printf("hello, world\n");
    return 0;
}
```

## Сборка и запуск

```bash
gcc hello.c -o hello
./hello
```

Ожидаемый вывод:

```
hello, world
```

## Команды, которые пригодились

| Команда | Действие |
|---|---|
| `mkdir lab-00` | создать каталог |
| `cd lab-00` | войти в каталог |
| `cd ..` | подняться на уровень выше |
| `pwd` | показать текущий путь |
| `ls` | список файлов |
| `nano hello.c` | редактор (Ctrl+O — сохранить, Ctrl+X — выйти) |
| `gcc hello.c -o hello` | компиляция |
| `./hello` | запуск |
| `rm hello` | удалить файл |
