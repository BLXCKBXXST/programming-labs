# Задание 1 — Форматированный вывод

Воспроизвести вид экрана из методички с использованием переменных и/или констант (нельзя зашивать готовую строку).

## Эталонный вывод

```
|555|
|     555|
|555     |

Words:       59
Letters:   1004
Digits:       8

a = 0.100000; b = 0.200000
a + b = 0.30000000000000004
```

## Идея

Между `%` и `d` (или `f`) ставится **число — ширина поля**:

| Спецификатор | Что делает | Пример при `555` |
|---|---|---|
| `%d`   | минимальная ширина | `555` |
| `%8d`  | ширина 8, выравнивание справа | `     555` |
| `%-8d` | ширина 8, выравнивание **слева** (знак `-`) | `555     ` |
| `%.2f` | вещественное, 2 знака после точки | `0.30` |
| `%.17f`| 17 знаков — видно ошибку округления `double` | `0.30000000000000004` |

## Почему `0.1 + 0.2 ≠ 0.3`

`double` хранится в формате IEEE-754 (двоичная плавающая точка). Числа `0.1` и `0.2` имеют **бесконечные** двоичные представления, поэтому округляются. После сложения округлений накапливается погрешность ≈ 2⁻⁵². Спецификатор `%.17f` показывает достаточно цифр, чтобы её увидеть.
