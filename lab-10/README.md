# Лабораторная работа №10 — Структуры: арифметика обыкновенных дробей

Тип `Fraction` на основе структуры, умножение и деление дробей, вывод результата.

## Структура

```
lab-10/
├── fraction.h      # Объявление структуры Fraction и прототипы функций
├── fraction.c      # Реализация: make/mul/div/print
├── script.c        # main: ввод двух дробей, вывод произведения и частного
└── README.md
```

## Сборка и запуск

```
gcc script.c fraction.c -o lab-10
./lab-10
```
