# РГР-15 — Игра «Змейка» (C++)

**Вариант 15** | Язык: C/C++ | Дисциплина: Программирование

## Сборка

### Linux / macOS (с ncurses)
```bash
g++ -std=c++17 -O2 -o snake snake.cpp -lncurses
./snake
```

### Windows (MinGW)
```bash
g++ -std=c++17 -O2 -o snake.exe snake.cpp
snake.exe
```
> На Windows используется `<conio.h>` + `<windows.h>` — ncurses не нужен.

## Файлы

```
RGR-15/
├── snake.cpp      — исходный код C++17
├── config.cfg     — настройки/параметры игры
└── README.md      — описание
```

## Параметры config.cfg

| Параметр     | Умолч. | Описание                      |
|--------------|--------|-------------------------------|
| cols         | 22     | Ширина поля                   |
| rows         | 22     | Высота поля                   |
| speed_easy   | 200    | Задержка (мс) — лёгкий        |
| speed_medium | 120    | Задержка (мс) — средний       |
| speed_hard   | 60     | Задержка (мс) — сложный       |
| food_score   | 10     | Базовые очки за еду           |
| init_length  | 3      | Начальная длина змейки        |

## Управление

| Клавиша    | Действие               |
|------------|------------------------|
| ↑ ↓ ← →    | Движение змейки        |
| P / Space  | Пауза / Продолжить     |
| R          | Рестарт                |
| ESC        | Выход в меню           |
| Q (меню)   | Выход из игры          |

## Реализованные критерии

- ✅ Движение в 4 направлениях (стрелки)
- ✅ Увеличение змейки при съедании еды
- ✅ Границы поля (столкновение = конец игры)
- ✅ Столкновение с собой = конец игры
- ✅ Настройки из файла `config.cfg`
- ✅ Текущий счёт + рекорд в HUD
- ✅ Рекорд сохраняется в `records.dat` (отдельно для каждого уровня)
- ✅ 3 уровня сложности с разными скоростями и множителями очков
