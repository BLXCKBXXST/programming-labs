# РГР-15 — Игра «Змейка»

**Вариант 15** | Дисциплина: Программирование

## Описание

Консольная игра «Змейка» в браузере (HTML5 Canvas).

### Реализованные критерии

| Критерий | Реализация |
|---|---|
| Движение в 4 направлениях | Стрелки ↑ ↓ ← → |
| Увеличение при съедании еды | Хвост растёт, счёт увеличивается |
| Границы поля | Столкновение со стеной = конец игры |
| Столкновение с собой | Конец игры при наезде на хвост |
| Настройки из файла | `config.json` — поле, скорость, очки |
| Текущий счёт | HUD в верхней части |
| Рекорд | Отдельный рекорд для каждой сложности |
| Уровни сложности | Лёгкий / Средний / Сложный |

## Файлы

```
RGR-15/
├── snake_game.html   — основная игра (один файл, запуск в браузере)
├── config.json       — настройки/параметры игры
└── README.md         — описание
```

## Запуск

Открыть `snake_game.html` в любом современном браузере.

## Управление

| Клавиша | Действие |
|---|---|
| `↑ ↓ ← →` | Движение змейки |
| `P` / `Space` | Пауза / Продолжить |
| `R` | Рестарт |

## Настройки (config.json)

```json
{
  "field": { "cols": 25, "rows": 25, "cell_size": 22 },
  "difficulty": {
    "easy":   { "speed_ms": 120, "score_mult": 1 },
    "medium": { "speed_ms": 70,  "score_mult": 2 },
    "hard":   { "speed_ms": 38,  "score_mult": 3 }
  },
  "game": { "food_score": 10, "initial_length": 3 }
}
```

## Начисление очков

- **Лёгкий**: 10 очков × 1 за еду
- **Средний**: 10 очков × 2 за еду
- **Сложный**: 10 очков × 3 за еду
