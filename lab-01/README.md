## ⚙️ Сборка

```bash

make all              # эквивалент make

make clean
```

## 🧠 Что отрабатывается

- `if / else if / else` цепочки и **порядок** проверок (см. високосный год: проверка на 400 идёт раньше, чем на 100).
- Чтение разнородного ввода `scanf("%lf%c", ...)` — число + символ единицы измерения.
- `tolower()` из `<ctype.h>` для нормализации регистра.
- Описание плоских областей системами неравенств (`&&` для пересечения).
- Подключение `<math.h>` и линковка `-lm`.

См. README в подпапках для деталей по каждой задаче.
