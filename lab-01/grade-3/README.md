# Оценка 3 — Високосный год

## Задача

Запросить у пользователя год и вывести `YES` / `NO` — високосный он или нет.

## Алгоритм

1. Если год делится на **400** без остатка → високосный.
2. Иначе если делится на **100** → НЕ високосный.
3. Иначе если делится на **4** → високосный.
4. Иначе → НЕ високосный.

Порядок проверок важен: 2000 делится и на 400, и на 100, и на 4. Без проверки на 400 первым 2000 был бы записан как «не високосный».

## Код

[leap.c](leap.c):

```c
if (year % 400 == 0)      leap = 1;
else if (year % 100 == 0) leap = 0;
else if (year % 4 == 0)   leap = 1;
else                       leap = 0;
```

## Сборка

```bash
gcc leap.c -o leap
./leap
```

## Тесты

| Ввод | Ожидаем | Почему |
|---|---|---|
| `2000` | YES | делится на 400 |
| `1900` | NO  | делится на 100, но не на 400 |
| `2024` | YES | делится на 4, не на 100 |
| `2023` | NO  | не делится на 4 |
