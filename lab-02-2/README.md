# Лабораторная работа №2.2 — Вложенные циклы

![C](https://img.shields.io/badge/C-11-blue) ![Build](https://img.shields.io/badge/build-Makefile-orange) ![Topic](https://img.shields.io/badge/topic-nested%20loops-yellow)

> Печать фигур из символов: квадрат из чисел, ступенчатая лесенка (вариант 3, по `25 % 22`), спираль улиткой.

## 📊 Структура и оценки

| Grade | Задача | Файл |
|---|---|---|
| 4 ★★ | Квадрат N×N из чисел | [grade-4/square.c](grade-4/square.c) |
| 5 ★★★ | Лесенка букв (вариант 3) | [grade-5/stairs.c](grade-5/stairs.c) |
| ★ | Спираль N×N (улитка) | [grade-star/spiral.c](grade-star/spiral.c) |

> «Для оценки 4 решаем только задачу на 4. Для оценки 5 — только задачу на 5» — поэтому каждая задача в своей папке, не вложены друг в друга.

## 🗂 Файлы

```
lab-02-2/
├── Makefile
├── grade-4/
│   ├── square.c          # квадрат N×N из чисел 1..N²
│   └── README.md
├── grade-5/
│   ├── stairs.c          # лесенка AAAAA / BBBB / CCC / DD / E
│   └── README.md
├── grade-star/
│   ├── spiral.c          # улитка N×N
│   └── README.md
└── README.md
```

## ⚙️ Сборка

```bash
make all           # собрать все три

make grade-4       # только квадрат
make grade-5       # только лесенка
make grade-star    # только спираль

make clean
```

## 🎨 Примеры вывода

### Квадрат (grade-4), N=3

```
1 2 3
4 5 6
7 8 9
```

### Лесенка (grade-5, вариант 3), N=5

```
AAAAA
BBBB
CCC
DD
E
```

### Спираль (grade-★), N=4

```
 1  2  3  4
12 13 14  5
11 16 15  6
10  9  8  7
```

## 🧠 Что отрабатывается

- Вложенные `for`-циклы (циклы в циклах).
- Регулируемая высота фигуры — параметр читается со `scanf`.
- Управление **четырьмя движущимися границами** (для спирали).
- Форматный вывод с динамической шириной поля `%*d` (для спирали).
