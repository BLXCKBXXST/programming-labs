# ★ Звёздочка — Спираль N×N

## Задача

Заполнить квадрат N×N числами 1..N² **по спирали (улиткой)** — идём по периметру по часовой стрелке, постепенно сворачивая внутрь.

## Примеры

```
N=4:
 1  2  3  4
12 13 14  5
11 16 15  6
10  9  8  7

N=6:
 1  2  3  4  5  6
20 21 22 23 24  7
19 32 33 34 25  8
18 31 36 35 26  9
17 30 29 28 27 10
16 15 14 13 12 11
```

## Алгоритм

Идея — четыре движущиеся границы `top, bottom, left, right`. На каждой итерации внешнего `while` пишем один «виток»:

1. **→** идём вправо по верхней строке `top` от `left` до `right`, потом `top++`.
2. **↓** идём вниз по столбцу `right` от нового `top` до `bottom`, потом `right--`.
3. **←** идём влево по строке `bottom`, потом `bottom--`.
4. **↑** идём вверх по столбцу `left`, потом `left++`.

Шаги 3 и 4 защищены проверками — без них при нечётном N последний центральный элемент запишется дважды.

## Код (ядро)

См. [spiral.c](spiral.c):

```c
int top = 0, bottom = n - 1, left = 0, right = n - 1;
int v = 1;
while (top <= bottom && left <= right) {
    for (int j = left;   j <= right;  j++) a[top*n + j]    = v++;  top++;
    for (int i = top;    i <= bottom; i++) a[i*n + right]  = v++;  right--;
    if (top <= bottom)
        for (int j = right; j >= left; j--) a[bottom*n + j] = v++,  bottom--;
    if (left <= right)
        for (int i = bottom; i >= top; i--) a[i*n + left]   = v++,  left++;
}
```

> Поле — динамический одномерный массив на N² элементов: `calloc((size_t)n*n, sizeof(int))`. Освобождается `free(a)`.

## Ширина поля при печати

Чтобы столбцы были ровные, ширина каждого числа — это число разрядов в `N²`:

```c
int width = 1;
for (int t = n * n; t >= 10; t /= 10) width++;
printf("%*d", width, val);
```

Спецификатор `%*d` берёт ширину из аргумента.

## Сборка и запуск

```bash
gcc spiral.c -o spiral
./spiral
# затем N (1..20)
```
