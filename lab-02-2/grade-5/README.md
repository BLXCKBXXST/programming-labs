# Оценка 5 — Лесенка (вариант 3 из 22)

> Студент №25. `25 % 22 = 3`, значит **вариант 3**.

## Задача (из методички)

```
AAAAA
BBBB
CCC
DD
E
```

На строке `i` (0-индекс) печатаем букву `'A' + i` ровно `N − i` раз. Высота — N.

## Код

[stairs.c](stairs.c):

```c
for (int i = 0; i < n; i++) {
    char ch = 'A' + i;
    int width = n - i;
    for (int j = 0; j < width; j++) {
        putchar(ch);
    }
    putchar('\n');
}
```

## Сборка и запуск

```bash
gcc stairs.c -o stairs
./stairs
# затем введи N (1..26)
```

## Тесты

| N | Вывод |
|---|---|
| `1` | `A` |
| `3` | `AAA` / `BB` / `C` |
| `5` | `AAAAA` / `BBBB` / `CCC` / `DD` / `E` |

Ограничение N ≤ 26 — иначе буквы кончатся (после `Z`).
