# Лабораторная работа №15

**Тема:** Работа с текстовыми и бинарными файлами  
**Вариант:** 21 — Кроссовки (sneakers)

## Предметная область

База данных сотрудников: имя, ID, уровень (Level_A / Level_B / Level_C).

## Структура

| Папка | Описание |
|-------|----------|
| `grade-3/` | Чтение CSV, вывод таблицы |
| `grade-4/` | CSV или DAT → бинарная БД, поиск по ID |
| `grade-5/` | CSV / DAT / RLE → конвертация между форматами, RLE-сжатие |

## Формат CSV

```
Имя;ID;Уровень
James Wilson;6001;Level_B
```

## Сборка и запуск

```bash
cd grade-N
make
./program <файл>
```

## Тестовые файлы

- `staff.csv` — основная база (10 записей)
- `staff2.csv` — дополнительная база для тестирования (12 записей)
