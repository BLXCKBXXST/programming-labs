# Лабораторная работа №15 — grade-5

## Описание

Программа загружает записи из файла `.csv`, `.dat` или `.rle` и предоставляет интерактивное меню для работы с данными.

## Структура проекта

```
grade-5/
├── main.c       — точка входа, определение типа файла
├── record.c     — загрузка, вывод, сохранение, сжатие
├── record.h     — структура Record, константы, прототипы
├── menu.c       — интерактивное меню
├── menu.h       — прототип run_menu
├── Makefile
├── staff.csv    — тестовый CSV (набор 1)
└── staff2.csv   — тестовый CSV (набор 2)
```

## Сборка

```bash
make
```

## Запуск

```bash
./program staff.csv
./program staff2.csv
./program database.dat
./program database.rle
```

## Меню

Меню формируется динамически в зависимости от типа загруженного файла:

| Пункт | CSV | DAT | RLE |
|-------|-----|-----|-----|
| 1. Поиск по ID | ✓ | ✓ | ✓ |
| 2. Показать все записи | ✓ | ✓ | ✓ |
| 3. Сохранить в DAT | ✓ | — | ✓ |
| 4. Сохранить в CSV | — | ✓ | ✓ |
| 5. Сжать в RLE | ✓ | ✓ | — |
| 0. Выход | ✓ | ✓ | ✓ |

## Формат CSV

Разделитель — `;`, без заголовка:

```
Название;ID;Категория
```

Пример (`staff.csv`):
```
Меч рыцаря;101;uncommon
Доспех дракона;102;rare
```
