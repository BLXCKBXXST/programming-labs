# grade-5

Расширенная работа с бинарной базой данных: поддержка CSV, DAT и RLE-сжатия.

## Сборка

```bash
make
```

## Запуск

```bash
./program <файл.csv|файл.dat|файл.rle>
```

## Поддерживаемые форматы

| Формат | Поведение |
|--------|-----------|
| `.csv` | Читает CSV, загружает в память |
| `.dat` | Читает бинарную БД напрямую |
| `.rle` | Декомпрессия → `database_restored.dat` → загрузка |

## Меню

| Пункт | Описание | Доступность |
|-------|----------|-------------|
| 1 | Поиск сотрудника по ID | всегда |
| 2 | Показать всех сотрудников | всегда |
| 3 | Сохранить в `database.dat` | `.csv` и `.rle` |
| 4 | Сохранить в CSV | `.dat` и `.rle` |
| 5 | Сжать в RLE | `.csv` и `.dat` |
| 0 | Выход | всегда |

## Примеры

```bash
./program staff.csv
./program database.dat
./program database.rle
```

## Файлы

| Файл | Описание |
|------|----------|
| `database.dat` | Бинарная БД |
| `database.rle` | RLE-сжатый файл |
| `database_restored.dat` | Восстановленный из RLE |
