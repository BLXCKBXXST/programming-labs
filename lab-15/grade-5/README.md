# grade-5

Расширенная работа с бинарной базой данных: поддержка CSV, DAT и RLE-сжатия. Включает весь функционал grade-4.

## Сборка

```bash
make
```

## Запуск

```bash
./program <файл.csv|файл.dat|файл.rle>
```

## Поддерживаемые форматы

| Формат | Поведение |
|--------|-----------|
| `.csv` | Читает CSV, загружает в память |
| `.dat` | Читает бинарную БД напрямую |
| `.rle` | Декомпрессия → `database_restored.dat` → загрузка |

## Меню

Пункты меню зависят от формата входного файла:

| Пункт | Описание | `.csv` | `.dat` | `.rle` |
|-------|----------|:------:|:------:|:------:|
| 1 | Поиск сотрудника по ID | ✓ | ✓ | ✓ |
| 2 | Показать всех сотрудников | ✓ | ✓ | ✓ |
| 3 | Сохранить в database.dat | ✓ | — | ✓ |
| 4 | Сохранить в CSV | — | ✓ | ✓ |
| 5 | Сжать в RLE (database.rle) | ✓ | ✓ | — |
| 0 | Выход | ✓ | ✓ | ✓ |

## Примеры

```bash
./program staff.csv
./program database.dat
./program database.rle
```

## Файлы

| Файл | Описание |
|------|----------|
| `staff.csv` | Основная тестовая база (15 записей) |
| `books.csv` | Дополнительный тестовый файл (8 записей) |
| `database.dat` | Бинарная БД (создаётся программой) |
| `database.rle` | RLE-сжатый файл (создаётся программой) |
| `database_restored.dat` | Восстановленный из RLE (создаётся программой) |
