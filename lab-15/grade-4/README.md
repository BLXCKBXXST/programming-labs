# grade-4

Работа с бинарной базой данных сотрудников.

## Сборка

```bash
make
```

## Запуск

```bash
./program <файл.csv|файл.dat>
```

## Поддерживаемые форматы

| Формат | Поведение |
|--------|-----------|
| `.csv` | Читает CSV, загружает в память, пункт «Сохранить в DAT» доступен |
| `.dat` | Читает бинарную БД напрямую |

## Меню

| Пункт | Описание | Доступность |
|-------|----------|-------------|
| 1 | Поиск сотрудника по ID | всегда |
| 2 | Показать всех сотрудников | всегда |
| 3 | Сохранить в `database.dat` | только `.csv` |
| 0 | Выход | всегда |

## Примеры

```bash
./program staff.csv
./program database.dat
```
