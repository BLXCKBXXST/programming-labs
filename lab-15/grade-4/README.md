# Оценка 4 — Бинарный файл и поиск по ID — сотрудники с полями Имя, ID, Уровень.

## Что делает программа

1. Читает `staff.csv` и выводит таблицу (всё из оценки 3)
2. Сохраняет массив структур в `database.dat` через `fwrite()`
3. Интерактивное меню с поиском сотрудника по ID через `fread()`

## Структура файлов

```
grade-4/
├── main.c        # точка входа
├── employee.c    # логика: CSV, fwrite, fread, поиск
├── employee.h    # структура Employee + объявления
├── menu.c        # интерактивное меню
├── menu.h
├── staff.csv
├── Makefile
└── README.md
```

## Сборка и запуск

```bash
make
./program staff.csv
```
