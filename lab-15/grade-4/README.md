# grade-4

Работа с бинарной базой данных сотрудников. Является основой для grade-5, которая добавляет поддержку RLE-сжатия.

## Сборка

```bash
make
```

## Запуск

```bash
./program <файл.csv|файл.dat>
```

## Поддерживаемые форматы

| Формат | Поведение |
|--------|-----------|
| `.csv` | Читает CSV, загружает в память |
| `.dat` | Читает бинарную БД напрямую |

## Меню

Пункты меню зависят от формата входного файла:

| Пункт | Описание | `.csv` | `.dat` |
|-------|----------|:------:|:------:|
| 1 | Поиск сотрудника по ID | ✓ | ✓ |
| 2 | Показать всех сотрудников | ✓ | ✓ |
| 3 | Сохранить в database.dat | ✓ | — |
| 4 | Сохранить в CSV | — | ✓ |
| 0 | Выход | ✓ | ✓ |

## Примеры

```bash
./program staff.csv
./program database.dat
```
