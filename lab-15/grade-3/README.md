# Оценка 3 — Чтение текстового CSV-файла — сотрудники с полями Имя, ID, Уровень.

## Что делает программа

1. Принимает имя CSV-файла через аргумент командной строки
2. Читает файл построчно через `fgets()`
3. Разбивает поля по `;` через `strtok()`
4. Выводит отформатированную таблицу в консоль

## Структура файлов

```
grade-3/
├── main.c       # чтение CSV и вывод таблицы
├── staff.csv    # входные данные
├── Makefile
└── README.md
```

## Формат staff.csv

```
Elon Musk;5001;Level_A
Jeff Bezos;5002;Level_B
```

## Сборка и запуск

```bash
make
./program staff.csv
```

## Пример вывода

```
Загружено сотрудников: 15

№     Имя                      ID       Уровень
---   ------------------------ -------- ----------
1     Elon Musk                5001     Level_A
2     Jeff Bezos               5002     Level_B
...
```
