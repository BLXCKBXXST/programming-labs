# Lab 12 — sneakers — Оценка 3

## Что реализовано

Всё находится в одном файле `main.c`.

### 1. Структура данных

```c
typedef struct {
    char brand[20];  // бренд кроссовок
    char model[20];  // модель
    int  size;       // размер
    int  price;      // цена
} Sneaker;
```

### 2. Массив

```c
#define N 100
Sneaker sneakers[N];
```

Статический массив из `N` элементов, размер задан через `#define`.

### 3. Заполнение случайными данными — `FillSneakers()`

- `brand` — случайно из осмысленных: `Nike`, `Adidas`, `Puma`
- `model` — случайно из: `Pro`, `Super`, `Ultra`
- `size` — целое число в диапазоне `[36..44]`
- `price` — осмысленная цена зависит от бренда/модели (200–700)

### 4. Вывод в виде таблицы — `PrintSneakers()`

Функция выводит все поля структуры с выровненными столбцами через `%-Ns` форматы `printf`.

### 5. Алгоритм сортировки — `BubbleSortByPrice()` (сортировка пузырьком)

Сортировка по целочисленному полю `price`. Алгоритм:

- внешний цикл `i` от `0` до `N-1`
- внутренний цикл `j` от `0` до `N-1-i`
- если `sneakers[j].price > sneakers[j+1].price` — меняем элементы местами
- на каждом проходе внешнего цикла самый дорогой элемент «всплывает» в конец массива
- сложность: O(n²)

### 6. Вывод до и после сортировки

```c
FillSneakers(sneakers);
printf("ДО сортировки:"); PrintSneakers(sneakers);
BubbleSortByPrice(sneakers);
printf("ПОСЛЕ сортировки по цене:"); PrintSneakers(sneakers);
```

## Сборка и запуск

```bash
gcc -Wall main.c -o sneakers
./sneakers
```
