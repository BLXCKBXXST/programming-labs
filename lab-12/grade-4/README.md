# Lab 12 — Grade 4

Разбито на 3 файла. Статическая библиотека собирается через Makefile.

## Структура
```
grade-4/
├── src/
│   ├── main.c
│   └── sneakers.c
├── include/
│   └── sneakers.h
└── Makefile
```

## Сборка
```bash
make
./sneakers
make clean
```

## Смысл статической библиотеки
Статическая библиотека (`.a`) — это архив из объектных файлов (`.o`).
При линковке весь код встраивается прямо в бинарник — не нужно никаких внешних файлов для запуска.
Но если библиотека меняется — нужно пересобирать всю программу целиком.
