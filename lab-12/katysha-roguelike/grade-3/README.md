# Lab 12 — katysha-roguelike — Оценка 3

## Что реализовано

Всё находится в одном файле `main.c`.

### 1. Структура данных

Используется вариант — комната Roguelike-игры:

```c
typedef struct {
    char name[50];   // имя комнаты
    int  level;      // уровень/сложность комнаты
    int  number;     // номер комнаты
    int  resolution; // размер комнаты
} Room;
```

### 2. Массив

```c
#define N 100
Room rooms[N];
```

Статический массив из `N` элементов структуры `Room`, размер задан через `#define`.

### 3. Заполнение случайными данными — `FillRooms()`

- `name` — выбирается случайно из осмысленных названий: `Dungeon`, `Crypt`, `Throne`, `Library`, `Armory`, `Garden`, `Cave`, `Tower`, `Vault`, `Chapel`
- `level` — целое число в диапазоне `[1..10]`
- `number` — порядковый номер (от 1 до N)
- `resolution` — целое число в диапазоне `[5..50]`

### 4. Вывод в виде таблицы — `PrintRooms()`

Функция выводит все поля структуры с выровненными столбцами через `%-Ns` форматы `printf`.

### 5. Алгоритм сортировки — `BubbleSortByLevel()` (сортировка пузырьком)

Сортировка по целочисленному полю `level`. Алгоритм:

- внешний цикл `i` от `0` до `N-1`
- внутренний цикл `j` от `0` до `N-1-i`
- если `rooms[j].level > rooms[j+1].level` — меняем элементы местами
- на каждом проходе внешнего цикла самая сложная комната «всплывает» в конец массива
- сложность: O(n²)

### 6. Вывод до и после сортировки

```c
FillRooms(rooms);
printf("ДО сортировки:"); PrintRooms(rooms);
BubbleSortByLevel(rooms);
printf("ПОСЛЕ сортировки по уровню:"); PrintRooms(rooms);
```

## Сборка и запуск

```bash
gcc -Wall main.c -o rooms
./rooms
```
