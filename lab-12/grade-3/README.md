# Lab 12 — Grade 3

Всё в одном .c файле: структура, массив, заполнение, вывод, сортировка пузырьком.

## Сборка и запуск
```bash
gcc -Wall main.c -o sneakers
./sneakers
```

## Алгоритм пузырька
На каждом проходе внешнего цикла сравниваются соседние элементы, и больший из них "всплывает" вправо.
После N-1 проходов массив отсортирован. Сложность O(n²).
