# Lab 13 — Grade 3

Структура проекта с Makefile и статической библиотекой.

## Структура
```
grade-3/
├── src/
│   ├── main.c
│   └── sneakers.c
├── include/
│   └── sneakers.h
├── Makefile
└── .gitignore
```

## Сборка
```bash
make        # собрать проект (цель all)
./sneakers  # запустить
make clean  # удалить build/ и бинарник
```
