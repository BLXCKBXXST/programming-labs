# Lab 14 — Grade 5

Два ряда разной длины (N и K). Навигация WASD включая переход между рядами.

## Схема списка
```
S → [a1] → [a3] → ... → [aN] → nil   (верхний ряд, N элементов)
       ↓        ↓              ↓
     [a2] ← [a4] ← ...        nil   (нижний ряд, K элементов, обратный)
```

Если N > K — лишние узлы верхнего ряда имеют `cross = NULL`.

## Сборка
```bash
gcc -Wall main.c -o list
./list 6 4
```

## Управление
- `D` / `6` — вправо в своём ряду
- `A` / `4` — назад в своём ряду
- `S` / `2` — вниз (верхний → нижний)
- `W` / `8` — вверх (нижний → верхний)
- `R` — вернуться к S
- `Q` — выход
