# Lab 14 — sneakers — Оценка 5

## Что реализовано

Включает всё из оценок 3 и 4, плюс:

### 7. Два ряда разной длины (N и K)

Схема списка (зигзаг):

```
S → [a1] → [a3] → [a5] → ... → [aN] → nil    (верхний ряд, N элементов)
         ↕       ↕      ↕              ↕
       [a2] ← [a4] ← [a6] ← ... ← [aK]          (нижний ряд, K элементов)
```

Структура узла:

```c
typedef struct Node {
    Sneaker     *data;
    struct Node *next;   // верх: вправо | низ: влево
    struct Node *cross;  // переход между рядами (вверх/вниз)
    int          row;    // 0 = верхний, 1 = нижний
} Node;
```

Узлы длинного ряда, выходящие за пределы короткого, получают `cross = NULL`.

### Навигация

| Клавиша | Действие |
|---|---|
| `D` / `6` | вправо по верхнему / влево по нижнему ряду |
| `A` / `4` | влево по верхнему / вправо по нижнему ряду |
| `S` / `2` | вниз (через `cross`) |
| `W` / `8` | вверх (через `cross`) |
| `R` | вернуться к S (начало верхнего ряда) |
| `Q` | выход |

Если перемещение невозможно (`cross == NULL` или `next == NULL`) — выводится сообщение из-за разных длин.

## Сборка и запуск

```bash
gcc -Wall main.c -o sneakers
./sneakers 6 4    # N=6 (верхний), K=4 (нижний)
```
