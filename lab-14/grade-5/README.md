# Lab 14 — Grade 5

Два ряда разной длины (N и K). Навигация WASD включая переход между рядами.

## Схема списка
```
S → [a1] → [a3] → ... → [aN-1] → nil
       ↓        ↓               ↓
     [a2]  ←  [a4] ← ...  ← [aN]
     nil      nil             nil
```

- верхний ряд: `next` идёт вправо (a1 → a3 → ...)
- нижний ряд: `next` идёт влево (aN → ... → a4 → a2 → nil), у каждого узла nil справа
- `cross`: верхний ↕ нижний по позиции
- если N ≠ K — лишние узлы длинного ряда имеют `cross = NULL` (п.б7)

## Сборка
```bash
gcc -Wall main.c -o list
./list 6 4
```

## Управление
- `D` / `6` — вправо (верх: по next; низ: через cross)
- `A` / `4` — назад (верх: через обход; низ: по next влево к a2)
- `S` / `2` — вниз (верхний → нижний через cross)
- `W` / `8` — вверх (нижний → верхний через cross)
- `R` — вернуться к S
- `Q` — выход
