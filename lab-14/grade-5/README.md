# Лабораторная работа №14 — Оценка 5

## Что делает программа

Программа строит **двухрядную связную структуру** из двух списков:
- **Верхний ряд** из `N` узлов (обход `next` — слева направо)
- **Нижний ряд** из `K` узлов (обход `next` — справа налево)

Ряды связаны вертикально через указатель `cross`.  
Так как `N` и `K` могут отличаться, у «лишних» узлов длинного ряда `cross = NULL`.

---

## Схема структуры

```
S → [1] → [2] → [3] → NULL      (верхний ряд, next вправо)
     |      |     |
    [1] ← [2] ← [3] ← [4] → NULL  (нижний ряд, next влево)
```

Здесь N=3, K=4. Узел [4] нижнего ряда не имеет пары → его `cross = NULL`.

---

## Ключевые понятия

### Почему нижний ряд хранится «задом наперёд»?

Нижний ряд добавляется через `node->next = botHead` (в начало).  
Это значит, что обход по `next` идёт **вправо→влево**.  
Зато проще добавлять: не нужен хвостовой указатель.

### Что такое cross?

`cross` — вертикальная связь между рядами.  
`upper->cross = lower` и `lower->cross = upper`.  
Если у узла нет пары (длины разные) — `cross = NULL`.  
Это и есть выполнение пункта 7 задания.

### Как двигаться вправо по нижнему ряду?

Нижний `next` указывает влево. Поэтому «вправо» делаем через верхний ряд:
```
cur → cross (подняться вверх) → next (шаг вправо) → cross (опуститься вниз)
```
Если на каком-то шаге `NULL` — конец.
