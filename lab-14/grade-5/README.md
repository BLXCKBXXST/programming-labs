# Lab 14 — sneakers — Оценка 5

## Что реализовано

### Зигзаг-список (два ряда + cross-указатели)

```
S → [a1] → [a3] → ... → [aN-1] → nil   (верхний ряд, next →)
       ↓       ↓              ↓
     [a2]  ← [a4] ← ... ← [aN]          (нижний ряд, next ←)
```

```c
typedef struct Node {
    Sneaker     *data;
    struct Node *next;   // верх: вправо | низ: влево
    struct Node *cross;  // вверх/вниз (NULL если без пары)
    int          row;    // 0 = верх, 1 = низ
} Node;
```

### Навигация WASD

| Клавиша | Действие |
|---|---|
| `D` / `6` | вправо по текущему ряду |
| `A` / `4` | влево по текущему ряду |
| `S` / `2` | вниз через cross |
| `W` / `8` | вверх через cross |
| `R` | вернуться к S |
| `Q` | выход |

### Запуск

```bash
gcc -Wall main.c -o sneakers
./sneakers 5 4   # верх: 5 узлов, низ: 4
./sneakers       # ввод через scanf
```
