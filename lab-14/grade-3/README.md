# Лабораторная работа №14 — Оценка 3

## Что делает программа

Программа создаёт **односвязный список** из `n` узлов.  
Каждый узел хранит информацию о кроссовке:
- бренд (`brand`)
- модель (`model`)
- размер (`size`)
- цена (`price`)

Данные заполняются случайно. Список выводится на экран.

---

## Ключевые понятия

### Что такое односвязный список?

Это набор узлов, где каждый узел знает только про **следующий**:

```
[Nike Pro] → [Adidas Super] → [Puma Ultra] → NULL
```

`NULL` в конце — значит «дальше ничего нет».

### Почему `Node **head` (двойной указатель)?

Функция `PushBack` должна **изменить** переменную `head` в `main`.  
Чтобы функция могла менять чужую переменную, передаём **адрес этой переменной** — то есть `&head`.  
Внутри функции пишем `*head = ...`, чтобы изменить то, на что указывает адрес.

### Почему `malloc`?

Заранее не известно, сколько узлов будет. `malloc` выделяет память **во время работы программы** ровно столько, сколько нужно.  
После использования память нужно освободить через `free`.

## Сборка и запуск

```bash
gcc main.c -o lab14_3
```

Расшифровка флагов:
- `main.c` — исходный файл
- `-o lab14_3` — название выходного файла (без этого будет `a.out`)

Запуск с вводом n с клавиатуры:
```bash
./lab14_3
```

Запуск с передачей n как аргумента:
```bash
./lab14_3 5
```
