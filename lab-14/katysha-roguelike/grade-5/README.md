# Lab 14 — katysha-roguelike — Оценка 5

## Что реализовано

### Зигзаг-список (два ряда + cross-указатели)

```
S → [aN] → [aN-2] → ... → [a2] → nil   (верхний ряд, чётные)
       ↕          ↕              ↕
     [aN-1] → [aN-3] → ... → [a1] → nil  (нижний ряд, нечётные)
```

```c
typedef struct Node {
    Room        *data;
    struct Node *next;   // вправо по ряду
    struct Node *cross;  // вверх/вниз между рядами (NULL если без пары)
    int          row;    // 0 = верхний, 1 = нижний
} Node;
```

### Навигация WASD

| Клавиша | Действие |
|---|---|
| `D` / `6` | вправо по текущему ряду |
| `A` / `4` | влево по текущему ряду |
| `S` / `2` | вниз через cross |
| `W` / `8` | вверх через cross |
| `R` | вернуться к S (начало верхнего ряда) |
| `Q` | выход |

### Запуск

```bash
gcc -Wall main.c -o rooms
./rooms 5 4   # верхний ряд 5 узлов, нижний 4
./rooms       # ввод через scanf
```
