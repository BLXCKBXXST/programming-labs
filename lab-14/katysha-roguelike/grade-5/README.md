# Lab 14 — katysha-roguelike — Оценка 5

## Что реализовано

Включает всё из оценок 3 и 4, плюс:

### 7. Два ряда разной длины (N и K)

Схема списка (вариант):

```
S → [aN] → [aN-2] → [aN-4] → ... → [a2] → nil     (верхний ряд, N узлов)
      ↕          ↕           ↕              ↕
    [aN-1] → [aN-3] → [aN-5] → ... → [a1] → nil    (нижний ряд, K узлов)
```

- **Верхний ряд** — N узлов с чётными номерами: aN, aN-2, ..., a2. `next` → вправо.
- **Нижний ряд** — K узлов с нечётными номерами: aN-1, aN-3, ..., a1. `next` → вправо.
- **`cross`** — связь между i-м узлом верхнего и i-м узлом нижнего ряда.
- Узлы длинного ряда, выходящие за пределы короткого, получают `cross = NULL`.

Структура узла:

```c
typedef struct Node {
    Room        *data;
    struct Node *next;   // вправо по ряду
    struct Node *cross;  // переход между рядами
    int          row;    // 0 = верхний, 1 = нижний
} Node;
```

### Навигация

| Клавиша | Действие |
|---|---|
| `D` / `6` | вправо по текущему ряду (`next`) |
| `A` / `4` | влево по текущему ряду |
| `S` / `2` | вниз (через `cross`) |
| `W` / `8` | вверх (через `cross`) |
| `R` | вернуться к S (начало верхнего ряда) |
| `Q` | выход |

Если `cross == NULL` (ряды разной длины) — выводится сообщение о невозможности перехода.

## Сборка и запуск

```bash
gcc -Wall main.c -o rooms
./rooms 5 4    # N=5 (верхний), K=4 (нижний)
./rooms        # ввод с клавиатуры
```
