# Lab 14 — sneakers — Оценка 4

## Что реализовано

Включает всё из оценки 3, плюс:

### 4. Указатель на структуру в узле

Данные теперь хранятся через **указатель**, а не встроенно в узел.
Функция `CreateNode()` инициализирует этот указатель через `CreateSneaker()`.

```c
typedef struct Node {
    Sneaker     *data;   // указатель на структуру (malloc)
    struct Node *next;
    struct Node *prev;   // указатель назад
} Node;
```

### 5. Двусвязный список + указатель S на начало

```
S → [1] ⇆ [2] ⇆ [3] ⇆ ... ⇆ [N] → nil
```

- `head` — указатель `S` (начало списка)
- `PushBack(Node **head, Node **tail)` — добавляет в конец, проставляет `next`/`prev`

### 6. Интерфейс WASD / 4, 6

| Клавиша | Действие |
|---|---|
| `D` / `6` | перемещение вперёд (`next`) |
| `A` / `4` | перемещение назад (`prev`) |
| `Q` | выход |

При прижатии `next == NULL` (конец) — предлагается вернуться в начало (S).При `prev == NULL` (начало) — выводится сообщение об этом.

## Сборка и запуск

```bash
gcc -Wall main.c -o sneakers
./sneakers 8
```
