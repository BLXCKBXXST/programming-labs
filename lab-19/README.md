# Lab 19 — Консольный банкомат

Учебный проект на C++17: консольный банкомат с картами, счетами,  
бинарным журналом транзакций и восстановлением после сбоя.

## Структура

```
lab-19/
├── CMakeLists.txt
├── README.md
└── atm/
    ├── main.cpp            # точка входа
    ├── card.cpp/h          # класс Card (номер, PIN-хэш, блокировка)
    ├── account.cpp/h       # баланс, снятие/пополнение, история
    ├── transaction.h/cpp   # struct Transaction (тип, сумма, дата)
    ├── journal.cpp/h       # append-only бинарный журнал
    ├── recovery.cpp/h      # восстановление баланса из журнала
    ├── atm_engine.cpp/h    # главная логика (сессия, меню, лимиты)
    ├── file_storage.cpp/h  # чтение/запись карт и счетов
    └── tests/
        ├── doctest.h       # загружается через CMake FetchContent
        ├── test_card.cpp
        ├── test_account.cpp
        └── test_journal.cpp
```

## Сборка

```bash
cd lab-19
cmake -B build -DCMAKE_BUILD_TYPE=Debug
cmake --build build
```

## Запуск

```bash
./build/atm
```

При первом запуске автоматически создаётся папка `data/` с демо-данными:

| Карта              | PIN  | Баланс      |
|--------------------|------|-------------|
| 1234567890123456   | 1234 | 50 000 руб  |
| 9876543210987654   | 5678 | 10 000 руб  |

## Запуск тестов

```bash
cmake --build build
ctest --test-dir build --output-on-failure
```

## Логика восстановления

При каждом запуске `Recovery::restore()` читает `data/journal.bin`  
и воспроизводит все транзакции карты, устанавливая актуальный баланс.  
Частичные записи в конце файла (при краше в момент записи) молча игнорируются.
